# University Database System

## Overview
This project focuses on designing and implementing a relational database system to manage students, instructors, courses, and enrollments.

## Features
- Designed a normalized database schema
- Designed EER diagrams and a relational schema to model system structure and relationships
- Implemented SQL queries and constraints to ensure data integrity

## Technologies
- MySQL
- SQL

- schema.sql: database structure
- queries.sql: sample queries
- EER_diagram.png: database design
